- Tool: OpenAI image generation

## Prompt

Create a high-resolution product UX concept sheet for Axis Envoys inside a
premium personal operating system. Show: a compact Envoy sprite beside a
truthful background-work status card; a multi-task switcher; explicit waiting,
approval-required, failed, and completed/result states; Focus, Intel, and Ask as
capability tabs independent of appearance; an Interface Studio quick picker;
and a full Envoy Lab with owner library, animation-state preview, private
generation history, cost/allowance, cancellation, and failed-stage retry.

Include desktop dark mode, mobile collapsed status button, and warm light-mode
insets. Use cold graphite/navy glass, warm bone text areas, precision hairlines,
restrained user-accent brass, semantic status icons plus labels, stable compact
geometry, strong keyboard focus, and clear real actions. Show indeterminate
stages and real count forms, never invented percentages. Use abstract placeholder
text blocks rather than readable copy or nonsense words.

Avoid chatbot-first layout, cartoon speech bubbles, terminal faces, robots,
floating orbs, excessive glow, purple gradients, fake online dots, decorative
meters, nested glass-card soup, and status conveyed only by color/animation.
