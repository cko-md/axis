# Navigation Status

AXIS navigation separates production modules from modules that are still in beta or lab state.

## Production

Production modules support the core workflow and should stay in the primary Daily/System navigation:

- Command
- Dispatch
- Schedule
- Agenda
- Mail
- Notes
- Control Room

## Beta

Beta modules are usable but still need deeper workflow hardening, provider parity, or persistence coverage:

- Objectives
- Debrief
- Pipeline
- Literature
- People
- Briefing
- Fund

## Labs

Labs modules may contain partial, static, or exploratory workflows and must not appear equally production-ready:

- Vitality
- Atelier
- Listening Vault
- Library
- Supper Club

## Promotion Rule

Move a module to production only after its primary workflow supports list, detail, action, persistence, feedback, empty/error states, and relevant provider/Sentry validation.
